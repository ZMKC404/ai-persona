# 🧬 你是 AI 时代的什么人？

> 一个面向 AI 创业者的 16 型人格测试。12 道题，4 个维度，测你在 AI 创业里是哪种物种。

## ✨ 特性

- 📱 **手机端优先**，单手即可完成
- ⚡ **单文件 HTML**，零依赖，零构建
- 🎨 **一键生成海报**，Canvas 现场渲染
- 🔗 **URL 直链人格**（如 `#ADSO`），便于朋友对号入座
- 🥚 **隐藏人格彩蛋**：全选极端答案解锁「VC 观察员 👁️」
- ⚔️ **对比模式**：输入朋友代号生成关系分析
- 📋 **炫耀文案**：一键复制到 bio / 朋友圈 / X

## 🚀 部署（60 秒上线）

1. Fork 本仓库，或创建新仓库并上传 `index.html`
2. Settings → Pages → Source 选 `main` branch / `/ (root)`
3. 等 1 分钟，访问 `https://<你的用户名>.github.io/<仓库名>/`
